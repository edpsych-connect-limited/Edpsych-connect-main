# EdPsych Connect: The Comprehensive Ecosystem Map

## 1. The Core Engine (Identity & Access)
*   **Multi-Tenant Architecture:** Secure data isolation for Schools, LAs, and MATs.
*   **Role-Based Access Control (RBAC):** Distinct interfaces for Super Admins, Teachers, SENCOs, Parents, and Professionals.
*   **Onboarding Flows:** Tailored signup experiences for each user type.
*   **Dashboard:** Personalized landing pages with "At a Glance" metrics (e.g., "Pending Assessments", "Active Interventions").

## 2. The Educational Suite (The "Work")
*   **EHCP Manager:**
    *   Digital EHCP creation and versioning.
    *   Workflow tracking (Draft -> Review -> Final).
    *   Outcome monitoring.
*   **Assessment Engine:**
    *   Standardized tests (Phonics, Emotional Literacy).
    *   Automated scoring and report generation.
    *   Dynamic Assessment tools.
*   **Intervention Designer:**
    *   Bespoke plan creation (Cognition, SEMH, Communication).
    *   Strategy library (now expanding with your "Training Content").
    *   Progress monitoring (Pre/Post measures).
*   **Case Management:**
    *   Centralized student profiles.
    *   Document storage.
    *   Chronological history of support.

## 3. The Innovation Engine (The "Future")
*   **"Developers of Tomorrow" Coding Curriculum:**
    *   **Tracks:** Block Coding (Beginner), Python (Intermediate), React (Advanced).
    *   **Teacher Dashboard:** Assign modules, track class progress, identify struggling students.
    *   **Gamified Learning:** XP, Levels, Unlockable content.
    *   **Accessibility:** Voice guides, high-contrast modes.
*   **AI Agents:**
    *   **Report Writer:** Drafts professional reports from raw data.
    *   **Plan Generator:** Suggests interventions based on assessment scores.
    *   **Chatbot:** "Help Centre" AI for instant platform support.

## 4. The Engagement Layer (Gamification)
*   **Battle Royale:** Competitive quizzes to reinforce learning.
*   **Economy System:** Earn "Merits" for completing tasks (lessons, assessments).
*   **Avatar System:** Customizable profiles to build student identity.
*   **Leaderboards:** School-wide and class-wide engagement tracking.

## 5. The Professional Growth Hub (CPD)
*   **Training Centre:**
    *   Video-based courses.
    *   Quizzes and Certification.
    *   **NEW:** Massive library integration (ADHD, ASD, Trauma, etc.).
*   **Marketplace:**
    *   Buy/Sell resources.
    *   Peer-reviewed content.
*   **Research Hub:**
    *   Access to evidence-based papers.
    *   "Research to Practice" summaries.

## 6. The Community Bridge
*   **Parent Portal:**
    *   Real-time progress views.
    *   Secure messaging with school.
    *   Resource library for home support.
*   **Professional Networking:**
    *   Forums for SENCOs and EPs.
    *   Case discussion boards (anonymized).
*   **Institutional Management:**
    *   LA-level analytics (trends across schools).
    *   Resource allocation tools.

## 7. The "Safety Net"
*   **Safeguarding:** Secure logging of concerns.
*   **GDPR Compliance:** Data encryption, consent management.
*   **Accessibility Suite:** WCAG 2.1 compliance tools (Text-to-Speech, Dyslexia font).
