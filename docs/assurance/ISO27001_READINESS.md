# ISO/IEC 27001 Readiness (Internal)

ISO 27001 is an organisational certification. Code quality alone is insufficient: the key is an operating **ISMS**.

## What we can self-audit internally

### 1) ISMS fundamentals
- [ ] Scope statement (systems, people, locations)
- [ ] Information security policy set
- [ ] Asset inventory and classification
- [ ] Risk assessment + treatment plan
- [ ] Statement of Applicability (SoA)

### 2) Operational controls
- [ ] Access control policy + access reviews
- [ ] Secure SDLC and change management
- [ ] Logging/monitoring and incident response
- [ ] Supplier management and subprocessors
- [ ] BCP/DR tested

### 3) Audit machinery
- [ ] Internal audit schedule and reports
- [ ] Corrective action tracking
- [ ] Management review records

## What external auditors will expect

- Evidence that controls operate over time (not a one-off snapshot)
- Traceability: risk -> control -> evidence
