# Role Journeys and Success Criteria

This document captures the primary journeys and success criteria per role to
support navigation, onboarding, and governance readiness.

## Platform Owner (SUPER_ADMIN)
Primary journeys
- Governance and compliance oversight via `/admin` and `/admin/ethics`
- Verify statutory workflow health across EHCP and case tracking
- Manage access controls and platform configuration
- Review analytics and delivery performance

Success criteria
- Compliance dashboard reviewed and evidence snapshot exported
- Owner-only actions restricted to SUPER_ADMIN
- Key operational risks acknowledged and tracked

## Platform Admin (ADMIN)
Primary journeys
- Access operational governance in `/studios/admin`
- Review analytics and reporting outputs
- Monitor case and workflow throughput
- Coordinate cross-team delivery actions

Success criteria
- Admin operations accessible without owner-only controls
- Reports and analytics reviewed within cadence
- Workflow bottlenecks surfaced and logged

## Local Authority
Primary journeys
- Triage statutory deadlines on `/la/dashboard`
- Review compliance risk and EHCP modules
- Approve or progress EHCP requests
- Track outcomes and escalations

Success criteria
- Statutory timelines within policy thresholds
- At-risk cases identified before breach
- EHCP approvals documented with evidence

## Educational Psychologist (EP)
Primary journeys
- Start and complete assessments in `/assessments`
- Generate structured EHCP evidence and reports
- Assign evidence-based interventions
- Manage cases and collaborate with schools

Success criteria
- Assessments completed with professional interpretation
- Reports generated and stored with audit trace
- Interventions assigned with monitoring plan

## School Leadership (SENCO / School Admin)
Primary journeys
- Track referrals and case readiness
- Deploy interventions and monitor progress
- Share progress data with stakeholders
- Coordinate training and CPD

Success criteria
- Referrals progressed to appropriate support
- Interventions show measurable progress signals
- Progress updates shared on schedule

## Teacher / Staff
Primary journeys
- Set up class groups and baseline checks
- Assign interventions and daily supports
- Monitor progress and report outcomes

Success criteria
- Classroom plans created and tracked weekly
- Interventions delivered with fidelity tracking
- Progress logged for each student

## Parent / Carer
Primary journeys
- Access child profile and progress updates
- Review goals, milestones, and support plans
- Collaborate with staff via messaging

Success criteria
- Progress visibility maintained without barriers
- Feedback shared and acknowledged by staff
- Support updates received in a timely manner

## Student
Primary journeys
- Start learning missions and progress activities
- Track achievements and rewards
- Engage in gamified learning modules

Success criteria
- Mission completion tracked with feedback
- Progress streaks visible and updated
- Rewards recorded without errors

## Researcher
Primary journeys
- Launch studies and manage ethics submissions
- Access anonymized data and datasets
- Monitor evidence quality and publish insights

Success criteria
- Ethics approvals tracked before data access
- Datasets accessed via approved channels
- Findings linked to evidence sources

## Default Professional
Primary journeys
- Run assessments and generate reports
- Deploy interventions
- Manage cases and inquiries

Success criteria
- Core workflows available without admin access
- Evidence captured for each workflow
- Inquiries responded to within SLA
