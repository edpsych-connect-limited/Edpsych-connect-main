# Voice Assistant Commands

The EdPsych Connect Voice Assistant allows you to navigate the platform hands-free.

## Activation
Click the microphone icon in the bottom right corner to start listening. The icon will pulse red when active.

## Supported Commands

| Command | Action |
|---------|--------|
| "Go to dashboard", "Open dashboard" | Navigates to the main dashboard |
| "Go to settings", "Open settings" | Opens the settings page |
| "Go to blog", "Read blog" | Opens the blog section |
| "Go home", "Open home" | Returns to the landing page |
| "Login", "Sign in" | Navigates to the login page |
| "Help" | Reads out the list of available commands |

## Feedback
The assistant will speak back to confirm your command (e.g., "Navigating to dashboard").
