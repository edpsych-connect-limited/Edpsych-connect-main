## 📂 Platform Sections Overview

### 1. Admin Dashboard (`/admin`)
Your primary control centre as SUPER_ADMIN:
- **Overview:** Institution subscription status, compliance metrics
- **Subscriptions:** Manage LA and school subscriptions
- **Analytics:** Platform usage, student outcomes, professional metrics
- **Reports:** Downloadable compliance and performance reports
- **Accounts:** User management across all roles
- **Compliance:** GDPR, ISO 27001, SOC 2 verification tools

### 2. LA Dashboard (`/la/dashboard`)
Your Local Authority management view:
- EHCP application tracking
- Statutory deadline compliance (20-week timeline)
- Caseworker workload management
- Professional panel coordination
- Multi-agency collaboration tools

### 3. Teacher Features (`/dashboard`, `/teachers`)
What teachers in Buckinghamshire schools would see:
- **Classroom Cockpit:** Real-time class management
- **Automatic Differentiation:** AI-powered lesson adaptation for 40 students
- **Student Profiles:** Automatically built from assessment data
- **Progress Tracking:** Individual and cohort analytics

### 4. Assessments (`/assessments`)
The ECCA Framework (EdPsych Connect Cognitive Assessment):
- Stealth assessments (game-based, anxiety-reducing)
- Dynamic cognitive profiling
- Evidence-based standardised tools
- Automatic report generation

### 5. Interventions (`/interventions`)
Evidence-based intervention library:
- Matched to assessment profiles
- Progress monitoring
- Research-validated approaches
- SEND-specific strategies

### 6. EHCP Tools (`/ehcp`)
Complete EHCP lifecycle management:
- Application initiation
- Professional contributions coordination
- Draft generation with AI assistance
- 20-week timeline tracking
- Annual review automation

### 7. Marketplace (`/marketplace`)
Professional services marketplace:
- Find Educational Psychologists
- LA Panel approved professionals
- Enhanced DBS and £6M insurance verified
- Direct booking system

### 8. Training Centre (`/training`)
CPD and professional development:
- ECCA Framework certification
- SEND awareness courses
- Assessment administration training
- Compliance and ethics modules

### 9. Help Centre (`/help`)
User support resources:
- Getting started guides
- Video tutorials
- FAQ articles
- Support ticket system
