# Fluency Interventions

**Speaker:** Dr. Scott Ighavongbe-Patrick
**Duration:** 70 seconds
**Audience:** Teachers, TAs

---

## Script

How do we build fluency? Repeated reading.
It sounds boring, but it works.

Have the child read the same short passage three times.
First time: Focus on accuracy.
Second time: Focus on smoothness.
Third time: Read it like an actor.

This 'over-learning' helps move words from the slow decoding pathway to the fast automatic pathway.
