# Teaching Phonics Systematically

**Speaker:** Dr. Scott Ighavongbe-Patrick
**Duration:** 75 seconds
**Audience:** Teachers

---

## Script

English is a complex code. But it is a code.
Systematic phonics means we teach the code in a logical order.

We start with the simple sounds (s, a, t, p).
Then we move to digraphs (sh, ch, th).
Then complex vowel teams (ai, ay, a-e).

We don't just throw random words at them. We give them the keys to unlock the code, one by one.
And we review, review, review. Mastery, not just exposure.
