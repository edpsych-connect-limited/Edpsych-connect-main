# Multisensory Teaching

**Speaker:** Dr. Scott Ighavongbe-Patrick
**Duration:** 65 seconds
**Audience:** Teachers, TAs

---

## Script

We learn best when we use more than one sense. For a dyslexic child, this is critical.
Multisensory teaching isn't just 'fun'. It's about creating multiple memory hooks.

When we teach the sound /a/, we don't just say it.
We see the letter 'a'.
We say the sound /a/.
We trace the letter in the air or on sandpaper.

This is VAKT: Visual, Auditory, Kinesthetic, Tactile.
It anchors the learning. If the auditory pathway is weak, the tactile pathway supports it.
