# The 8 Executive Functions

**Speaker:** Dr. Scott Ighavongbe-Patrick
**Duration:** 30 seconds

---

## Script

Dr. Russell Barkley calls ADHD Executive Function Deficit Disorder.
 
 It affects:
 1. Inhibition (Stopping yourself)
 2. Working Memory (Holding info in mind)
 3. Emotional Control
 4. Focus (Starting and sustaining)
 5. Planning
 6. Organisation
 7. Time Management
 8. Self-Monitoring
 
 When a student forgets their homework, it's not disrespect. It's a working memory failure.
 When they blurt out an answer, it's an inhibition failure.
 
 We need to support these functions, not punish the lack of them.
