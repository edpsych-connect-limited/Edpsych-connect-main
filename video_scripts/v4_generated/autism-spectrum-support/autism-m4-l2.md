# Structure and Predictability

**Speaker:** Dr. Scott Ighavongbe-Patrick
**Duration:** 60 seconds
**Audience:** Teachers

---

## Script

Anxiety thrives in uncertainty. The autistic brain is a prediction machine that hates surprises.

Structure provides safety.

This doesn't mean you have to be a robot. It means having clear routines.
"We always put our bags here." "We always start with a starter activity."

If there is a change—a supply teacher, a room swap—pre-warn. Use a social story. "Today is different, and that is okay."

When the environment is predictable, the brain can stop scanning for threats and start learning.
