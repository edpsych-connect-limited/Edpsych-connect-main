# Universal Accessibility

**Speaker:** Dr. Scott Ighavongbe-Patrick
**Duration:** 23 seconds

---

## Script

Accessibility isn't an afterthought. It's the foundation.
 Open the Accessibility Panel.
 
 You can change the font to OpenDyslexic.
 You can adjust the contrast for visual stress.
 You can use the screen reader for auditory processing.
 
 We believe in 'Universal Design'. By making the platform accessible for those with the highest needs, we make it better for everyone.
