**NARRATION:**
> "Before we begin observing, let's understand what we're assessing. The ECCA Framework covers four interconnected cognitive domains.

> First, Working Memory and Information Processing - this includes verbal and visual memory, processing capacity, and crucially, learning potential.

> Second, Attention and Executive Function - covering sustained attention, selective attention, and executive control.

> Third, Processing Speed - looking at cognitive efficiency, response speed, and automaticity.

> Finally, Learning and Memory - examining encoding strategies, retrieval processes, and learning patterns.

> Each domain uses a strengths-based approach and incorporates dynamic assessment principles."

**TECHNICAL NOTE:** Highlight each domain as mentioned, show brief preview of indicators

**[2:00-3:30] Domain-by-Domain Observation - Working Memory Example**

**VISUAL:** Click into Working Memory domain, scroll through indicators

**NARRATION:**
> "Let's look at the Working Memory domain in detail. You'll see specific behavioral indicators to observe. For example: 'Follows multi-step oral instructions' or 'Recalls details from stories or information presented verbally'.

> Here's how the test-teach-retest methodology works. First, observe Jake's initial performance on a digit span task. Let's say he recalls 4 digits forward, below the age-expected 6-7. This is the TEST phase.

> Next, the TEACH phase. We explicitly teach Jake a strategy - perhaps chunking numbers into pairs. We give him time to practice this new strategy.

> Finally, RETEST. We observe if Jake can now recall more digits using the strategy. If he improves to 6-7 digits, he's demonstrating good learning potential and responsiveness to intervention. This is far more valuable than just knowing his initial score."
